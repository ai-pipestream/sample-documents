# Climate Forecast Documents Test Samples

This folder contains sample NetCDF files for testing ClimateForcastMetadata extraction.

## Expected File Types:
- **NetCDF**: .nc (Network Common Data Form)

## Test Coverage:
- CF Convention metadata
- NetCDF dimensions and variables
- Temporal and spatial metadata
- Scientific data properties
- Model and instrument information
- Quality control and uncertainty

## Sample Files Needed:
- `climate_model.nc` - Climate model output
- `weather_data.nc` - Meteorological observations
- `ocean_data.nc` - Oceanographic measurements
- `atmospheric.nc` - Atmospheric research data
- `ncar_sample.nc` - NCAR CCSM format
