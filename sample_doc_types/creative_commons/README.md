# Creative Commons Documents Test Samples

This folder contains sample files with Creative Commons licensing for testing CreativeCommonsMetadata extraction.

## Expected File Types:
- **Images**: .jpg, .png (with CC metadata in EXIF/XMP)
- **Documents**: .pdf, .html (with CC license info)
- **Media**: .mp3, .mp4 (with CC licensing)

## Test Coverage:
- Creative Commons license types
- XMP Rights metadata
- Attribution requirements
- Usage permissions and restrictions
- DRM and rights management
- License validation and compliance

## Sample Files Needed:
- `cc_by_image.jpg` - Image with CC BY license
- `cc_by_sa_document.pdf` - Document with CC BY-SA
- `cc0_public_domain.png` - Public domain image
- `cc_licensed_audio.mp3` - Audio with CC license
- `rights_managed.pdf` - Document with rights info
