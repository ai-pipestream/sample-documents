# RTF Documents Test Samples

This folder contains sample RTF files for testing RtfMetadata extraction.

## Expected File Types:
- **RTF**: .rtf (Rich Text Format)

## Test Coverage:
- RTF-specific properties
- Creation and modification info
- Application metadata
- Formatting information
- Embedded objects

## Sample Files Needed:
- `simple_document.rtf` - Basic RTF with metadata
- `formatted_text.rtf` - Rich formatting
- `with_images.rtf` - RTF with embedded images
- `word_generated.rtf` - Created by Microsoft Word
- `legacy_format.rtf` - Older RTF version
