# Font Documents Test Samples

This folder contains sample font files for testing FontMetadata extraction.

## Expected File Types:
- **TrueType**: .ttf
- **Adobe Font Metrics**: .afm
- **OpenType**: .otf

## Test Coverage:
- Font family and PostScript names
- Technical metrics and properties
- Legal information (copyright, trademark)
- Character sets and encoding
- Font rendering properties

## Sample Files Needed:
- `sample_font.ttf` - TrueType font with metadata
- `adobe_font.afm` - Adobe Font Metrics file
- `opentype_font.otf` - OpenType format
- `system_font.ttf` - Common system font
- `custom_font.ttf` - Custom designed font
