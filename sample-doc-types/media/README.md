# Media Documents Test Samples

This folder contains sample media files for testing MediaMetadata extraction.

## Expected File Types:
- **Video**: .mp4, .avi, .mov, .mkv
- **Audio**: .mp3, .flac, .wav, .ogg

## Test Coverage:
- Duration and bitrates
- Codecs and technical specs
- XMP media metadata
- Artist/album information
- Video resolution and frame rate
- Audio sample rate and channels

## Sample Files Needed:
- `sample_video.mp4` - Video with metadata
- `music_track.mp3` - Audio with ID3 tags
- `high_quality.flac` - Lossless audio
- `short_clip.mov` - QuickTime format
- `podcast.wav` - Uncompressed audio
