# Image Documents Test Samples

This folder contains sample image files for testing ImageMetadata extraction.

## Expected File Types:
- **JPEG**: .jpg, .jpeg (with EXIF data)
- **TIFF**: .tif, .tiff
- **PNG**: .png
- **GIF**: .gif
- **BMP**: .bmp

## Test Coverage:
- EXIF data (camera settings, GPS coordinates)
- IPTC metadata (keywords, captions)
- Color profiles and technical specs
- Image dimensions and resolution
- Camera make/model information
- Geolocation data

## Sample Files Needed:
- `photo_with_exif.jpg` - Photo with rich EXIF data
- `gps_photo.jpg` - Photo with GPS coordinates
- `professional.tiff` - High-quality TIFF with metadata
- `transparent.png` - PNG with transparency
- `animated.gif` - Animated GIF
