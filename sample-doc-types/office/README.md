# Office Documents Test Samples

This folder contains sample Office documents for testing OfficeMetadata extraction.

## Expected File Types:
- **Microsoft Word**: .doc, .docx
- **Microsoft Excel**: .xls, .xlsx  
- **Microsoft PowerPoint**: .ppt, .pptx
- **OpenOffice/LibreOffice**: .odt, .ods, .odp

## Test Coverage:
- Creation/modification dates
- Authors and revision tracking
- Application info and version
- Page/word counts
- Document properties
- Embedded objects

## Sample Files Needed:
- `sample.docx` - Word document with metadata
- `sample.xlsx` - Excel spreadsheet with formulas
- `sample.pptx` - PowerPoint with slides and notes
- `sample.odt` - OpenOffice text document
- `legacy.doc` - Older Office format
