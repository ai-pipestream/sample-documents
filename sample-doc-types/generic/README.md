# Generic Documents Test Samples

This folder contains sample files of various formats for testing GenericMetadata extraction.

## Expected File Types:
- **WordPerfect**: .wpd
- **Quattro Pro**: .qpw, .wb3
- **PST**: .pst (Outlook data files)
- **Other**: Various uncommon formats

## Test Coverage:
- Basic document identification
- Format version detection
- Content analysis
- Technical metadata
- Parser-specific information
- Flexible struct-based metadata

## Sample Files Needed:
- `wordperfect_doc.wpd` - WordPerfect document
- `quattro_spreadsheet.qpw` - Quattro Pro file
- `outlook_data.pst` - PST mail archive
- `unknown_format.xyz` - Uncommon file type
- `legacy_document.old` - Very old format
