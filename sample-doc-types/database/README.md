# Database Documents Test Samples

This folder contains sample database files for testing DatabaseMetadata extraction.

## Expected File Types:
- **Access**: .mdb, .accdb (Microsoft Access)
- **Other**: Various database formats

## Test Coverage:
- Database schema information
- Table metadata and structure
- Creation dates and versions
- Database-specific properties
- Relationships and indexes

## Sample Files Needed:
- `sample_database.mdb` - Access database
- `modern_access.accdb` - Newer Access format
- `with_tables.mdb` - Multiple tables and relationships
- `empty_database.mdb` - Structure only
- `legacy_format.mdb` - Older database version
