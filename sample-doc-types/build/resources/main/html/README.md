# HTML Documents Test Samples

This folder contains sample HTML files for testing HtmlMetadata extraction.

## Expected File Types:
- **HTML**: .html, .htm
- **XHTML**: .xhtml

## Test Coverage:
- Meta tags and descriptions
- OpenGraph properties
- Twitter Cards metadata
- Dublin Core elements
- Geographic coordinates
- Structured data

## Sample Files Needed:
- `basic_page.html` - Simple HTML with meta tags
- `social_media.html` - Rich OpenGraph/Twitter Cards
- `dublin_core.html` - Dublin Core metadata
- `geo_tagged.html` - Geographic information
- `structured_data.html` - JSON-LD structured data
