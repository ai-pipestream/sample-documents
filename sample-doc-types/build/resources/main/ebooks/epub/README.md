# EPUB Documents Test Samples

This folder contains sample EPUB files for testing EpubMetadata extraction.

## Expected File Types:
- **EPUB**: .epub (Electronic Publication)

## Test Coverage:
- EPUB version and rendition layout
- Structural metadata (spine, manifest, TOC)
- Content analysis and chunking info
- Embedded resources (images, fonts, CSS)
- Navigation and accessibility
- DRM and encryption status

## Sample Files Needed:
- `simple_book.epub` - Basic EPUB with chapters
- `illustrated_book.epub` - EPUB with images
- `complex_layout.epub` - Advanced formatting
- `epub3_sample.epub` - EPUB 3.0 format
- `drm_protected.epub` - Encrypted EPUB (if available)
