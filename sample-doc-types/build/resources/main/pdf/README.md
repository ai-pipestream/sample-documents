# PDF Documents Test Samples

This folder contains sample PDF documents for testing PdfMetadata extraction.

## Expected File Types:
- **PDF**: .pdf (various versions)

## Test Coverage:
- PDF version and producer
- Encryption and security settings
- XFA forms and interactive elements
- Font information
- Page count and layout
- Metadata and properties
- Digital signatures

## Sample Files Needed:
- `simple.pdf` - Basic PDF with metadata
- `encrypted.pdf` - Password-protected PDF
- `form.pdf` - PDF with fillable forms
- `signed.pdf` - Digitally signed PDF
- `version_1_4.pdf` - Older PDF version
